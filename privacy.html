Privacy Policy for Clipper (Internal Chrome Extension)

Effective Date: 04 April 2025

1. Introduction

Clipper is a Chrome Extension developed exclusively for internal use. It is not distributed publicly and is intended to assist internal users in collecting and formatting information from web-based confirmation or checkout pages.

2. Information We Collect

The extension collects the following information:

  • Section, row, and seat information (if applicable)
  • Total quantity of items
  • Total price
  • Buyer name (entered manually)
  • Buyer email (entered manually)
  • Payment method used (selected manually)
  • Optional: event or item details, Customer ID, and Transaction ID

This data is stored locally in the browser using Chrome's local storage API. No data is shared externally.

3. How We Use the Information

The information is used only for internal workflow support, including:

  • Formatting and reviewing transaction details
  • Copying structured data to the clipboard
  • Exporting data as a CSV for internal use

4. Data Retention and Deletion

All data is stored locally and can be deleted via the extension’s “Clear Data” feature. No remote storage is involved.

5. Permissions

The extension requires the following permissions:

  • storage: Save local data
  • scripting, tabs: Extract page content
  • Host permissions: Access internal web environments as required

6. Security

Since no external transmission occurs, data security is primarily dependent on the user’s local device and browser settings.

7. Changes to This Policy

This policy may be updated periodically. All updates will be communicated internally.

8. Contact

For questions, please contact the internal team responsible for managing the extension.
